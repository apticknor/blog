# AnthonyTicknor.com

A theme for the [Metalsmith](http://www.metalsmith.io/) static site generator.

## Install & Build Steps

* `npm install`
* `node index.js`

## Meta Data

### All

* title: 
  * @description: itle of the content
  * @usage: prepended to `<title>`
  * @usage: display title on post detail and archive pages
